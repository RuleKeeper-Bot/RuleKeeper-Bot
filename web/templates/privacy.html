Privacy Policy for RuleKeeper

Last updated: April 11, 2025

RuleKeeper is an open-source Discord bot designed to assist with moderation, automation, and customization for Discord servers. Your privacy is important to us. This policy outlines what data RuleKeeper collects, how it's used, and your rights.

1. Data Collected

  • User and Message Metadata: Usernames, user IDs, message IDs, timestamps.
  • Server Configuration: Server IDs, channel IDs, role IDs, custom settings, and enabled features.
  • Moderation Logs: Records of moderation actions (e.g., kicks, bans, deleted messages).

RuleKeeper does not store message content unless explicitly configured for features like moderation logging or custom triggers.

2. How Data Is Used

  • To provide core features like moderation, logging, and custom command handling.
  • To manage settings and preferences on a per-server basis.
  • For diagnostic and debugging purposes when enabled by server admins.

3. Data Storage

  • Data is stored securely by the bot owner or hosting provider.
  • No data is sold, shared, or monetized.
  • If self-hosted, the responsibility for data handling falls to the instance operator.
